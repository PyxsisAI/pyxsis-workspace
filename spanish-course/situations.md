# 🌎 Situational Sessions — Everyday Survival Spanish

Real phrases for real places. Each session = key phrases + a mini-dialogue + a roleplay you can run with me (just text **"roleplay shopping"**, **"roleplay restaurant"**, etc.).

> Pronunciation reminder: vowels are pure — A=ah, E=eh, I=ee, O=oh, U=oo. **¿** and **¡** just signal a question/exclamation is coming.

---

## Universal phrases (work everywhere) ⭐
| Spanish | Pronunciation | English |
|---------|---------------|---------|
| Disculpe | dees-KOOL-peh | Excuse me (to get attention) |
| ¿Cuánto cuesta? | KWAN-toh KWES-tah | How much is it? |
| ¿Dónde está...? | DOHN-deh es-TAH | Where is...? |
| ¿Habla inglés? | AH-blah een-GLEHS | Do you speak English? |
| No entiendo | noh en-tee-EHN-doh | I don't understand |
| Más despacio, por favor | mahs des-PAH-see-oh | Slower, please |
| ¿Me puede ayudar? | meh PWEH-deh ah-yoo-DAR | Can you help me? |
| Solo estoy mirando | SOH-loh es-TOY mee-RAHN-doh | I'm just looking |

## Session 1 — 🛍️ Shopping
| Spanish | English |
|---------|---------|
| ¿Tiene esto en otra talla? | Do you have this in another size? |
| ¿Tiene otro color? | Do you have another color? |
| ¿Puedo probármelo? | Can I try it on? |
| Me lo llevo | I'll take it |
| ¿Aceptan tarjeta? | Do you accept card? |
| ¿Hay descuento? | Is there a discount? |
| Es muy caro | It's very expensive |
| Solo necesito una bolsa | I just need a bag |

**Mini-dialogue**
> **You:** Disculpe, ¿cuánto cuesta esto?
> **Clerk:** Veinte dólares.
> **You:** ¿Tiene otra talla? … Perfecto, me lo llevo. ¿Aceptan tarjeta?

## Session 2 — 🍽️ Restaurants
| Spanish | English |
|---------|---------|
| Una mesa para dos, por favor | A table for two, please |
| El menú, por favor | The menu, please |
| ¿Qué recomienda? | What do you recommend? |
| Quisiera... | I would like... (polite) |
| Para mí, el pollo | For me, the chicken |
| Sin cebolla, por favor | Without onion, please |
| ¿Tiene opciones vegetarianas? | Do you have vegetarian options? |
| La cuenta, por favor | The check, please |
| Estaba delicioso | It was delicious |
| Agua, por favor | Water, please |

**Mini-dialogue**
> **Waiter:** Buenas tardes, ¿una mesa?
> **You:** Sí, para dos, por favor.
> **Waiter:** ¿Qué desea?
> **You:** Quisiera el pollo, sin cebolla. Y agua, por favor.

## How to practice these
- **"roleplay shopping"** (or restaurant, café, directions, transport, pharmacy, emergency) → I play the clerk/waiter/stranger, you respond. I correct as we go.
- **"voice"** → say the phrases out loud in a voice note; I check pronunciation.
- Mix and match — real conversations jump between these. I'll throw curveballs once you're comfortable.
