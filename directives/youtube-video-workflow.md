# SOP: YouTube Video Production Workflow

Repeatable workflow for producing one faceless video. Target: 2 videos/week per active channel.
Tag: `#directive` · Related: [[MEMORY]] · [[USER]] · `channels/youtube-ideas/`

## Inputs
- Channel + topic (from `channels/youtube-ideas/`)
- Target length, CPM tier, affiliate angle (if any)

## Steps (maps to weekly schedule)

**Monday — Script**
1. Pull a topic from `channels/youtube-ideas/` backlog.
2. Draft script with ChatGPT/Claude. Hook in first 5 seconds; clear, actionable body; CTA at end.
3. Save script to `second-brain/documents/<channel>-<slug>-script.md`.

**Tuesday — Voice + Avatar**
4. Generate voiceover in ElevenLabs (consistent voice per channel).
5. Generate AI avatar/visuals in HeyGen or D-ID (if used).

**Wednesday — Edit**
6. Assemble in CapCut: VO + broll/avatar, captions, retention cuts (no dead air).

**Thursday — Package**
7. Thumbnail in Canva (high contrast, readable at small size).
8. Write SEO title + description + tags (TubeBuddy/VidIQ). Add affiliate links.

**Friday — Publish**
9. Schedule upload in YouTube Studio.
10. Log the video (title, link, status) and reply to comments.

## Outputs
- Published/scheduled video
- Script archived in `second-brain/documents/`
- Entry logged in `channels/youtube-ideas/` + a journal note in `second-brain/journal/`
